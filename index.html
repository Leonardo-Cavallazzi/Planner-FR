    Guida & condivisione

    Dove stanno i dati. Le tue tappe non sono su GitHub: GitHub tiene solo l’app. I dati vivono nel browser (salvataggio automatico su questo indirizzo) e, soprattutto, nel file .json o nel link/codice che generi tu. Quelli sono ciò che sposti tra dispositivi.

    Lavora su un solo indirizzo (il sito pubblico) per non avere “cassetti” separati tra localhost e sito.

    Link universale: pubblica index.html su GitHub Pages (Settings → Pages → branch main → Save). Otterrai un URL tipo https://tuonome.github.io/Planner-fr/: da lì “Copia link” produce un indirizzo che si apre ovunque.

    Migliorare la app senza perdere le tappe: ricarica solo index.html su GitHub; i dati (nel tuo .json/link) restano intatti grazie alla migrazione automatica di versione.
